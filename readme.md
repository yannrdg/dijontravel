Puis créez dans le dossiez php un fichier "config.php".

Remplacez les ..... par vos identifiants dans le fichier "config.php" :

<?php

/*** mysql hostname ***/

/*** mysql username ***/

/*** mysql password ***/

/*** dbname ***/
